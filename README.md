# flask & react

## how to use

- `python -m venv venv`
- win:`.\venv\Scripts\activate.bat` or mac:`. venv/bin/activate`
- `deactivate`

### frontend development

- `pip install -r requirements.txt`
- `flask run`
- `yarn`
- `yarn dev`

### backend development

- `yarn`
- `yarn build`
- `pip install -r requirements.txt`
- `flask run`
